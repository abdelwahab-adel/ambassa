## 📋 نبذة عن المشروع

**أمبيسا** هو موقع ويب سياحي شامل مخصص لمدينة أسوان بمصر. يقدم تجربة معرفية ثريّة عن تاريخ أسوان وثقافتها وطبيعتها الخلابة. الموقع يجمع بين التراث الفرعوني القديم والثقافة النوبية الأصيلة مع المعالم السياحية الحديثة.

### ✨ المميزات الرئيسية

- 🌐 **دعم لغتين**: العربية والإنجليزية مع تحويل سلس بين اللغتين
- 🏰 **معالم سياحية شاملة**: 12 معلم أثري وسياحي مع معلومات تفصيلية
- 🍽️ **مأكولات نوبية**: عرض الأطباق التقليدية المشهورة بأسوان
- 🚣 **أنواع رحلات متنوعة**: رحلات أثرية، نيلية، ثقافية، صحراوية، وعلاجية
- 💬 **مساعد ذكي**: روبوت دردشة ذكي (Ambesa) لتقديم المساعدة للزوار
- 📱 **تصميم متجاوب**: يعمل بسلاسة على جميع الأجهزة (ديسك، تابلت، موبايل)
- 🎨 **تصميم عصري وجذاب**: واجهة مستخدم حديثة مع تأثيرات بصرية متقدمة
- 🗺️ **خريطة تفاعلية**: موقع جغرافي محدد لمدينة أسوان

## 🛠️ التقنيات المستخدمة

### Frontend:
- **HTML5**: هيكل صفحات الويب
- **CSS3**: التصميم والرسوميات المتقدمة
- **JavaScript**: التفاعلية والديناميكية

## ⚙️ التثبيت والاستخدام

### الخطوة 1: استنساخ المشروع
```bash
git clone https://github.com/abdelwahab-adel/ambassa.git
cd ambassa
```

### الخطوة 2: فتح الموقع
يمكنك فتح `index.html` مباشرة في المتصفح، أو استخدام خادم محلي:

```bash
# استخدام Python
python -m http.server 8000

# أو استخدام Node.js (Live Server)
npx http-server
```

### الخطوة 3: الوصول للموقع
- محليًا: `http://localhost:8000`
- عبر الإنترنت: https://ambassa.vercel.app

## 🔧 التكوين والإعدادات

### إضافة مفتاح OpenRouter API
في `index.html` بحث عن:
```javascript
'Authorization': 'Bearer key_secret',
```
استبدل `key_secret` برمز API الفعلي من [OpenRouter](https://openrouter.ai)

### تخصيص الخريطة
عدّل رابط Google Maps Embed API حسب احتياجاتك في `index.html`
